Descrição

Neste projeto, obtive a experiência de desenvolver uma aplicação completa de CRM (Customer Relationship Management). Este foi o primeiro projeto profissional no qual eu fui o programdor responsável por desenvolver do inicio ao fim.

O objetivo dess aplicação é facilitar o contato com o cliente, atráves de ferramentas como: consultas de clientes personalizadas e especificas em vendas, contato e agendamentos com clientes via WhatsApp, e-mail e SMS, gráficos de valores de faturamente e comparações, análise de inadimplência e etc.

Nesta aplicação WEB foi utilizado o framework de Typescript SPA (single page application) Angular no front-end, juntamente com o Boostrap para estilização e responsividade.

No back-end, foi utilizado o framework da Microsoft: .NET com a linguagem C#, juntamente com o uso de ORMS como Entity Framework Core e Dapper ORM.

O back-end dessa aplicação foi criado utilizando os conceitos da arquitetura REST, portanto, foi desenvolvido uma API que compartilha dados com o frontend atráves de comunicação HTTP com os dados em formato JSON.

É importante salientar, que essa aplicação é constantemente atualizada, portanto, não é necessário especificar versões neste artigo.
