Bussiness App


Flutter Dart Firebase Python

Descrição


Este projeto, trata-se de um aplicativo Mobile para controle de finanças de empresas.
O objetivo desta aplicação é exibir em forma de valores, gráficos e percentuais os resultados e métricas de uma ou mais empresas em determinados períodos de tempos.

Este projeto, foi o primeiro produto Mobile de uma empresa, isto é, fui o desenvolvedor encarregado de criar a primeira experiência da empresa em uma nova plataforma. A linguagem, framework e banco de dados também foram tecnologias inovadoras dentro da empresa. A linguagem é o Dart, o framework é o Flutter e o banco de dados é o Firebase.

Este projeto possui diversos sistemas e funcionalidades, alguns deles são:

1. Fui o responsável por publicar o aplicativo nas duas principais lojas de aplicativos Mobile atualmente, App Store para IOS e Play Store para Android.

2. Somente clientes da empresa, podem criar contas e acessaor o aplicativo, a criação de contas é vinculada a uma integração com um sistema web, este sistema armazena os dados das contas de usuário dos clientes, sendo assim, para um usuário autenticar no aplicativo, é necessário ter uma conta no sistema web da empresa.
