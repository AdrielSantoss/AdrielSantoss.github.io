Descrição

Neste artigo irei, documentar minha experiência e conhecimento adquirido estudando sobre um conceito da área de Ciência de Dados, cujo sua sigla é: ETL (Extract, Transform, Load).

Para conhecimento teórico, consultei e pesquisei conteúdos sobre o tema na internet, dando prefêrencias para conteúdos academicos e de fontes confiáveis com referências externas. O objetivo dessa pesquisa é entender quais são os principais usos de uma operação ETL e quais as maneiras mais implementadas.

Para conhecimento prático, criei um script ETL simples com a linguagem R, somente com o uso de algumas bibliotecas auxiliares e de conversão. A ideia, foi criar um script que iria LER dados de uma fonte CSV (extract), percorrer estes dados modificando as suas estruturas (transform) e gravar estes dados em um novo formato de arquivo (load).
